Как запустить программу:
1. Открыть файл питон в пайчарме
2. Открыть терминал
3. Прописать в терминале pyuic5 main.ui -o main_ui.py
4. Нажать на стрелочку, которая запускает программу.
5. Ввести адекватные значения ( угол от 0 до 90 градусов, скорость от 0 до тысячи). БУКВЫ НЕ ВВОДИТЬ
6. Наслаждаться графиком
